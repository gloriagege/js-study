介绍call
call()方法在使用一个指定的this值和若干个指定的参数值的前提下调用某个函数或方法

eg
```
var foo = {
    value: 1
};

function bar() {
    console.log(this.value);
}

bar.call(foo); // 1
```
注意两点：
1.call改变了this的指向，指向到foo
2.bar函数执行了



###模拟实现第一步

试想当调用call的时候，把foo对象改造成如下
```
var foo = {
    value: 1,
    bar: function() {
        console.log(this.value)
    }
};

foo.bar(); // 1
```
此时this就指向了foo  
but  这样就给foo对象本身添加了一个属性  
但可以用delete再删除


so  模拟的步骤可以分为
1.将函数设为对象的属性
2.执行该函数
3.删除该函数

以上个例子为例  就是
```
// 第一步
foo.fn = bar
// 第二步
foo.fn()
// 第三步
delete foo.fn
```
